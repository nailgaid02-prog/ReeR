# ReeR - Безопасный VPN для обхода блокировок

Автоматизированное развертывание VPN-сервера на базе **Xray-core** для обхода блокировок YouTube, Instagram и других сервисов в РФ.

## ✨ Возможности

- 🚀 **Установка за 5 минут** - одна команда
- 🔒 **Безопасно** - используются официальные образы
- 📱 **Поддержка всех устройств** - iOS, Android, Windows, macOS, Linux
- 🌐 **Множество протоколов** - VLESS, VMess, Trojan, Shadowsocks
- 🎯 **Обходит DPI** - работает даже с глубокой проверкой пакетов
- 🖥️ **Веб-панель управления** - удобное добавление пользователей
- 📊 **Мониторинг** - статистика использования

## 🎯 Для чего это

- Доступ к заблокированным сервисам (YouTube, Instagram, др.)
- Безопасные запросы к AI API (OpenAI, Claude) из РФ
- Обход блокировок для всей семьи
- Работа с платёжными системами из-за границы

## 📋 Требования

- VPS сервер (Hetzner, DigitalOcean, Oracle Cloud и др.)
- Ubuntu 20.04+ / Debian 10+
- Минимум 1GB RAM, 1 CPU
- Открытые порты: 443, 2053, 10000

## 🚀 Быстрый старт

### 1. Подключитесь к серверу

```bash
ssh root@your-server-ip
```

### 2. Установите проект

```bash
git clone https://github.com/your-username/ReeR.git
cd ReeR
chmod +x setup.sh
./setup.sh
```

### 3. Получите доступы

После установки скрипт покажет:
- 🌐 Ссылку на веб-панель
- 👤 Логин и пароль администратора
- 📱 QR-коды для телефонов

## 📱 Подключение устройств

### iOS
1. Установите [Streisand](https://apps.apple.com/app/streisand/id6450534064) или [V2Box](https://apps.apple.com/app/v2box/id6446814690)
2. Отсканируйте QR-код или вставьте ссылку

### Android
1. Установите [v2rayNG](https://github.com/2dust/v2rayNG/releases)
2. Отсканируйте QR-код или вставьте ссылку

### Windows/macOS/Linux
1. Установите [Nekoray](https://github.com/MatsuriDayo/nekoray/releases)
2. Импортируйте конфигурацию

## 🔧 Расширенная настройка

### Изменить порты

Отредактируйте `docker-compose.yml`:

```yaml
ports:
  - "443:443"    # VLESS Reality
  - "2053:2053"  # VMess WebSocket
```

### Включить BBR (ускорение TCP)

```bash
./scripts/enable-bbr.sh
```

### Backup конфигурации

```bash
./scripts/backup.sh
```

## ❓ Частые вопросы

**Q: Как узнать IP сервера?**
```bash
curl ifconfig.me
```

**Q: Что делать если не работает?**
1. Проверьте логи: `docker-compose logs`
2. Проверьте firewall: `ufw status`
3. Перезапустите: `docker-compose restart`

**Q: Можно ли использовать свой домен?**
Да, отредактируйте конфиги в `configs/xray/config.json`

**Q: Сколько пользователей можно подключить?**
На сервере 1GB RAM - до 50 одновременных пользователей
